5 HTML tags I've learned

p
- This tag allows the programmer to write their own paragraph.
mark
- This tag allows the programmer to highlight specific text.
img
- This tag allows the programmer to add an image to their website, although it is also necessary to include the "src" and "alt" tags to do so.
table
- This tag allows the programmer to start making a table with information for their website.
ol
- This tag allows the programmer to add an ordered list to their website, although to actually include things in the list you must also use the "li" tag.
